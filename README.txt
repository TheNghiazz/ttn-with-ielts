Bản rút gọn — CHỈ dùng MyMemory (client-side)
==============================================
- Không cần backend (PHP/Python) hay Docker.
- Chạy trực tiếp bằng Live Server / Vite / http-server…

Sử dụng
-------
1) Mở thư mục này trong VS Code
2) Dùng Live Server (hoặc `npx http-server`), truy cập `http://localhost:...`
3) Vào tab Translate, nhập và xem kết quả — auto dịch khi gõ.

Lưu ý
-----
- MyMemory là dịch vụ công khai miễn phí → có thể rate-limit / độ chính xác không cao bằng API trả phí.
- Với nhu cầu ổn định hơn: bạn có thể quay lại phiên bản PHP or Python backend.
